Esse projeto tem por objetivo a construção de uma pokedex, que consome a [pokeAPI](https://pokeapi.co/)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Features

- Defaul image
- Shiny image
- stats
- types
- classes
- evolution stage
